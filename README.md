# Personal Introduction Program

## 📌 Project Overview
This is a simple Python program that collects user information and displays a friendly introduction message.

## 🎯 Objectives
- Learn how to take user input using input()
- Store data using variables
- Display output using print()
- Build a beginner-friendly interactive program

## ⚙️ Setup Instructions
1. Install Python (https://www.python.org/)
2. Download this repository
3. Open terminal or command prompt
4. Run the program:

```bash
python personal_intro.py
